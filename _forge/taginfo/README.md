# bukun_net


## 环境说明

需要 NLTK ， 还需要 nltk_data 。 这个可在线安装，但比较困难。
可在 github 单独下载，然后在程序中添加数据路径。

针对本项目，将 nltk_data 在当前文件夹解压缩。
